# Qui-a-cr-ce-visage-Deepfake-Model-Attribution-via-SWT-EfficientNet-B1
Source attribution of synthetic face images (GANs &amp; Diffusion) using Stationary Wavelet Transform (SWT) features + an EfficientNet-B1 classifier. Trains a 30-channel CNN to classify which generative model produced a face image (e.g., StyleGANs, DDPM, SDXL, etc.).
